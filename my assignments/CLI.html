Things I Learned

üåü dir /A reveals hidden files and directories, and you must use it to make real progress in the game.

üåü Tab auto-completion does not work for hidden paths‚Äîso if you're relying on it too much, you're going to miss things.

üåü File attributes like "read-only" can stop you from renaming or deleting files unless you change them first with attrib.

üåü Every directory matters‚Äîdon't assume a folder is empty just because dir looks that way. Always explore thoroughly.

üåü CLI commands are case sensitive in this environment, so you have to type file and folder names exactly.

üåü The tree command is super helpful when you're lost and want a visual map of where you've been.
